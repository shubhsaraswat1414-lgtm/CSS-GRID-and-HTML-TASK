A normal div arranges elements one below another by default, while a grid container allows elements to be placed in rows and columns.

The layout looks the same after applying Grid because no rows or columns are defined, so the browser keeps the default layout.

Yes, grid items are created automatically. Every direct child of a grid container becomes a grid item.

Grid can work without rows or columns, but without them there will be no visible change in the layout.

Grid needs defined rows, columns, or gaps to show visible changes in the layout.
