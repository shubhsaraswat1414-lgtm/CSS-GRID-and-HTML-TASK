Gap adds space between grid items, both between rows and between columns.

Gap is different from margin because it adds space only between grid items, while margin adds space around individual elements.

Gap does not increase the container size; it only creates space inside the container between items.

Yes, gap can be applied only to rows or only to columns using row-gap or column-gap.

Gap is preferred in Grid layouts because it is simple, clean, and keeps spacing consistent without breaking the layout.
